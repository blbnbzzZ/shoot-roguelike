# 法术系统设计方案
## Game Designer: GameDesigner

---

## 📋 设计概览

**系统名称**: 女巫法术系统  
**设计目标**: 丰富战斗深度，强化女巫主题，提供战术多样性  
**玩家体验**: "我是强大的女巫，掌握多种魔法"

---

## 🎯 设计支柱（Design Pillars）

1. **战术多样性** - 不同法术应对不同战况
2. **风险与奖励** - 强力法术需要管理资源（法力值）
3. **主题一致性** - 所有法术符合女巫幻想
4. **易学难精** - 简单上手，深度操作

---

## ⚡ 核心机制

### 1.1 法力值系统（Mana System）

**目的**: 限制法术滥用，创造资源管理决策  
**玩家感受**: 战略性地选择何时使用强力法术

```markdown
## Mechanic: 法力值系统

**Purpose**: 为法术系统提供资源限制，创造战术决策空间
**Player Fantasy**: 管理神秘能量，在关键时刻释放毁灭性魔法
**Input**: 
  - 施放法术时消耗法力
  - 自然恢复（每秒X点）
  - 拾取法力药水恢复
  
**Output**: 
  - 法力值变化（当前值/最大值）
  - UI反馈（法力条）
  - 法术可执行/不可执行状态
  
**Success Condition**: 玩家能在紧张战斗中做出"是否施放法术"的决策
**Failure State**: 法力值耗尽导致玩家无法使用法术，感到无力
**Edge Cases**:
  - 法力值刚好不足以施放法术时，显示"法力不足"提示
  - 法力恢复道具使用时，如果已满则无效化
  
**Tuning Levers**:
  - 法力上限（Base: 100, [PLACEHOLDER]）
  - 法力恢复速度（Base: 5/秒, [PLACEHOLDER]）
  - 各法术消耗（见下方法术列表）
  
**Dependencies**: 
  - Player.gd（添加法力值变量）
  - UI系统（添加法力条）
  - WeaponSystem.gd（集成法术施放逻辑）
```

**数值设计**:
```
Variable          | Base Value | Min  | Max  | Tuning Notes
------------------|------------|------|------|-------------------
玩家法力上限      | 100        | 50   | 200  | 随等级提升
法力恢复速度      | 5/秒       | 2/秒 | 10/秒| [PLACEHOLDER] - 测试战斗节奏
火球术消耗        | 20         | 10   | 40   | 中等消耗，常用法术
冰霜新星消耗      | 35         | 20   | 50   | 高消耗，应急法术
闪电链消耗        | 30         | 15   | 45   | 中等偏高，群体伤害
```

### 1.2 法术列表（Spell Catalog）

#### 🔥 火球术（Fireball）
- **施放方式**: 鼠标左键 + Q键（或右键）
- **伤害**: 50点（基础）
- **法力消耗**: 20点
- **冷却时间**: 无（受法力限制）
- **效果**: 发射火球，命中后爆炸，造成范围伤害
- **战术用途**: 中距离主力输出

**实现要点**:
```gdscript
## FireballSpell.gd
class_name FireballSpell
extends Resource

@export var display_name: String = "火球术"
@export var damage: float = 50.0
@export var mana_cost: float = 20.0
@export var explosion_radius: float = 50.0
@export var projectile_speed: float = 400.0
@export var sprite_texture: Texture2D
@export_multiline var description: String = "发射一枚火球，命中后爆炸造成范围伤害"
```

#### ❄️ 冰霜新星（Frost Nova）
- **施放方式**: 空格键 + Q（近身AOE）
- **伤害**: 30点（基础）
- **法力消耗**: 35点
- **冷却时间**: 8秒
- **效果**: 以自身为中心释放冰霜冲击波，减速敌人50%，持续3秒
- **战术用途**: 被包围时脱身，配合翻滚使用

#### ⚡ 闪电链（Chain Lightning）
- **施放方式**: 鼠标左键 + E
- **伤害**: 40点（基础） + 弹射伤害递减
- **法力消耗**: 30点
- **冷却时间**: 无
- **效果**: 攻击命中敌人后，闪电弹射到附近2个敌人，伤害递减20%
- **战术用途**: 清理群体敌人

#### 🛡️ 魔法护盾（Magic Shield）
- **施放方式**: Shift + Q（持续施放）
- **效果**: 消耗法力值维持护盾，吸收伤害
- **法力消耗**: 每秒10点
- **冷却时间**: 无（可持续施放）
- **战术用途**: 承受Boss攻击时使用

### 1.3 法术切换与快捷栏

**UI设计**:
```
快捷栏位置：屏幕底部中央
[1] [2] [3] [4]  - 数字键切换
鼠标滚轮 - 循环切换
Q/E/R/F - 快速施放
```

**法术获取方式**:
1. **拾取法术书** - 击败精英怪或Boss掉落
2. **奖励房兑换** - 用金币购买法术
3. **起始选择** - 每层地牢开始时三选一

## 📈 进度系统（Progression）

### 法术升级系统
每次拾取相同法术时，提供升级选项：

**火球术升级路径**:
- **等级1**: 基础伤害50
- **等级2**: 伤害60 + 点燃效果（持续伤害）
- **等级3**: 伤害70 + 爆炸半径+20%
- **等级4**: 伤害80 + 穿透1个敌人

**升级UI**:
```
┌─────────────────────────┐
│  法术升级！            │
│  [火球术] 已达到Lv2   │
│  □ 伤害+10            │
│  □ 法力消耗-5         │
│  □ 冷却-2秒           │
└─────────────────────────┘
```

## 📝 设计注释

**假设**:
- 玩家平均战斗时长30秒，法力值应支持2-3次法术施放
- 法术应比普通攻击强30-50%，但受法力限制
- 翻滚无敌帧0.25秒，冰霜新星可作为翻滚的替代策略

**待Playtest验证**:
- [ ] 法力恢复速度是否让玩家感到"差一点就能施放法术"
- [ ] 法术伤害是否足够让玩家感到"强力"
- [ ] 是否出现"法术OP导致普通攻击无用"的情况

## 🎨 视觉与音效需求

**视觉**:
- 火球术：橙色拖尾 + 爆炸粒子
- 冰霜新星：蓝色环形波 + 敌人变蓝（减速特效）
- 闪电链：黄色电弧 + 弹射连线

**音效**:
- 火球术：发射"嗖" + 爆炸"轰"
- 冰霜新星：冰晶碎裂音效
- 法力不足："失败音效"（低沉）
