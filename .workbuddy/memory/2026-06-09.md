## 子弹阻挡墙（隐形，只挡子弹）
- **Room.gd** 新增 `_bullet_barriers` 数组 + `_spawn_bullet_barriers()` 函数
- 在 `_ready()` 中调用，遍历所有 `doors` 子节点(Area3D)，在每个门口生成 StaticBody3D
- 关键参数: collision_layer=4(墙壁层), collision_mask=0, 无MeshInstance3D(完全隐形)
- 尺寸: 南北门 Vector3(80,200,10) / 东西门 Vector3(10,200,80), 高200覆盖地面到天花板

- 永久存在不受开门/关门影响

## Bug修复：大房间穿墙→出生位置错误→恶性循环
- **根因**: `_destroy_door_blockers()` 用 queue_free() 延迟删除 → 帧末才生效 → 窗口期内玩家挤压穿透 CharacterBody3D 允许的持续挤压后穿过门 → 没经过 _on_door_entered→_enter_room → 出生位置错误 → 之后每个房间都物理穿墙进入（恶性循环）
- **修复1 - 立即禁用碰撞**: `_destroy_door_blockers()` 先设 `collision_layer = 0` 再 queue_free()，消除窗口期
- **修复2 - PlayerDetector安全网**: `_on_player_enter` 增加判断：若房间已 `_started && _cleared`，直接 `_open_doors()` 而非重复 activate_room()，确保物理误入已清理房间后能正常离开
